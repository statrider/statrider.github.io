Privacy

STATrider does not send any private information from your PC to the cloud. Data will only be sent out from your PC when you use Events sheet to do a search.

Opening STATrider for the first time, you will see a security warning in a message bar as shown below.

In order to make STATrider fully functional you'll have to click Enable Content.

For more details about the security warning please visit Microsoft website at:

https://support.office.com/en-us/article/enable-or-disable-security-alerts-on-the-message-bar-adbf15d9-9195-4498-bdc3-5cb0dfe93852

STATrider uses a Microsoft Excel feature called Get & Transform to search events for you. In order to give you maximum transparency on how it is done, all M code used is unprotected.

For more information, please follow the link below:

https://support.office.com/en-us/article/get-transform-in-excel-881c63c6-37c5-4ca2-b616-59e18d75b4de
